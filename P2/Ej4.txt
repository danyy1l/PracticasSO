b) Hay que usar una variable global porque el handler solo puede recibir un entero, no una direccion ni nada que pueda compartir con el main. Por eso hay que usar una variable con el mismo scope que el handler y el main. Además se hace volatil para que el compilador no la optimice y se lea siempre de RAM, así como sig_atomic_t para que se modifique en un solo paso de procesador, haciendo que no se interrumpa su gestion por otra señal y estatica para que el scope sea unicamente de ese archivo
